# 导出与外部内容

导出（Export）
- SVG：`exportToSvg` 与 `getSvgString` 生成矢量输出，UI 可用于复制/下载。（相关函数在 exports 目录）
- 位图：`getSvgAsImage` 将 SVG 转为位图；`toImage/toImageDataUrl` 提供 Blob/DataURL（packages/editor/src/lib/editor/Editor.ts:9420 附近）。

外部内容（External Content）
- 统一入口：`putExternalContent({ type: 'files' | 'url' | 'text' | 'svg' | ... })`（定义在 Editor 周边类型中）。
- 能力：拖拽/粘贴文件、URL 解析成书签/Embed、SVG 导入、跨实例复制粘贴；内置校验/去重/尺寸约束。
- 资产：与 `@tldraw/assets` 和 workers 配合，进行图片优化/哈希去重/延迟上传。

设计要点
- 导出路径需与相机/可见性一致：如裁剪/透明度/分辨率/帧选择等参数要与实际画面一致。
- 外部内容必须经过校验与安全处理（MIME/尺寸/白名单），避免注入风险与 OOM。
